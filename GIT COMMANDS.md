## GIT BRANCH COMMANDS
> in Terminal (main)
  - create new branch
      git checkout -b <new branch name>
        ** -b is for "branch"
  - switch to/from branch
      git checkout <branch name>
  - push new branch
      git push -u origin <branch name>
        ** -u is for "upstream"
  - delete remote branch
      git push <branch name> -d <branch name>
  - delete local branch
      git branch -d <branch name>
